# Furnishings_Shop
Managment tool for furnishings shop
1.button Open File - choice excel file withs orders from local computer
2.button Show Data - show all data from file in new window
3.button Choice ID - show choiced order data in otdel`nii window . ID of order for choice put in text box raspolojenii bottom button Choice ID.
4.button Add Item - open new window for input data of new item.
5.button Remove Item- remove choiced in data grid window from dataset.
6.button Accept Changes - accept changes for excel file.
